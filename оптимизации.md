# Основные изменения и упрощения

- Удаление SDL2: Весь графический функционал заменен на простой вывод в консоль с использованием управляющей последовательности \033[H для обновления экрана на месте.

- Прямая логика: Функция get_neighbors теперь обрабатывает выбор целей более примитивно, без сложных математических вычислений смещения, что делает код прозрачным.

- Сохранение результатов: Добавлена работа с файлом population.csv, куда записывается состояние популяций на каждом шаге, выполняя требование методички о фиксации динамики.

- Оптимизация памяти: Матрица sea сделана статической, а структура cell избавлена от лишних типов данных, что минимизирует накладные расходы при обходе.
